## Installation
```bash
# Clean the dependencies
$ flutter clean

# Get the dependencies
$ flutter pub get
```

## Running the app
 Execute this commands combinations
```bash
# Select the device
$ Ctrl + Shift + P

# Select the option:
$ Flutter: Select Device
```
  Then select the device where you want to run the app
 ```bash
# Run the app
$ f5
```
